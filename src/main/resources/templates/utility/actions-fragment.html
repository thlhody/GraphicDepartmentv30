Emergency Information
Emergency Reset: Clears all caches and performs a complete system reset. Use this if you're experiencing persistent issues.

Force Refresh: Refreshes all user data from files without clearing caches. Use this if data seems outdated.

Quick Actions: These operations are designed to resolve common issues quickly without requiring a full application restart.
